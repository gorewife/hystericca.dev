Grimkeeper

discord bot for blood on the clocktower

GitHub add to discord

overview

grimkeeper handles voice channel automation, spectator following, timers, and grimoire tracking for blood on the clocktower discord communities.

features

  • automatic nickname prefixes for storytellers and spectators
  • voice channel cap management
  • shadow follow system for spectators
  • timer system with persistence
  • script polling for player voting
  • player tracking and grimoire management

commands

storyteller

*st — claim storyteller role
*call — move everyone to town square
*timer 5m — schedule delayed call
*poll 123 — create script voting poll
*night / *day — announce phase

spectator

*spec @user — shadow follow a player
*unspec — stop following
*dnd — prevent being followed

everyone

*! — toggle spectator mode
*players — list active players
*help — show all commands

tech stack

Python 3.8+ discord.py asyncio JSON persistence
